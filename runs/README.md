# QA Runs

Each folder under `runs/` represents one OpenClaw alpha/beta validation run.

Expected folder shape:

```text
runs/<tag>/
  run.json
  release-context.md
  freshness-report.md
  qa-checklist.md
  adversarial-review.md
  tester-instructions.md
  tester-status.json
  tester-status.md
  evidence-manifest.md
  triage-notes.md
  upstream-issues-filed.md
  closeout-report.md
  receipts/
```
